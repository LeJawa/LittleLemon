This website allows you to test the API of the Little Lemon restaurant.

Follow the links in the navigation bar to test the different endpoints.

This is not meant to be a complete test of the API, but rather a demonstration of the different endpoints.

It is recommended that you start with the Users page, as you will need users to authenticate to the other endpoints.

On the different pages, you will be testing the following endpoints:

  • Users:
    • /auth/users/
  • Menus:
    • {% url 'show_menu-list' %}
    • {% url 'manage_menu-list' %}
  • Bookings:
    • {% url 'user_booking-list' %}
    • {% url 'staff_booking-list' %}

Additionally, you will also use the following authentication endpoints:

    • /auth/token/login/
    • /auth/token/logout/

The endpoints can also be accessed using any REST client (e.g. Insomnia).

You can use the browser to reach the endpoints, but be aware that SessionAuthentication has been disabled.
